The next thing to creating simple website is a navigation. The Navigation allows for the user to navigate to different pages within the website. Also, all content that you write in html goes inside the body (<body></body>) tag.

This is my navigation that shows the different pages connected to the website.

          
            <body>
              <nav>
                <ul>
                  <li><a href="index.html">Home</a></li>
                  <li><a href="first.html">Header</a></li>
                  <li><a href="second.html">Navigation</a></li>
                  <li><a href="third.html">Content</a></li>
                </ul>
              </nav>
            </body>

In this piece of html we have the nav tag which tells the computer that this part is the navigation. Inside the nav we have the unordered list tag or ul. The ul places the items in a unordered list. Inside the ul, we have list items or li. This allows us to add item to the unordered list.
